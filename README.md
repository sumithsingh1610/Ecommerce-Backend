# E-Commerce Backend System

## Description
This is a backend system for an e-commerce platform built with Java, Spring Boot, and MySQL. It includes REST APIs for user authentication, product management, cart system, and order processing.

## Features
- User registration and login with JWT authentication
- Product management APIs (Add, Update, Delete, List products)
- Cart and Order management
- Scalable backend design

## Technologies & Skills
Java, Spring Boot, MySQL, REST API, JWT, Docker, Redis, Swagger

## How to Run
1. Clone the repository
2. Build with Maven/Gradle
3. Run the Spring Boot application
4. Use Postman to test APIs

## Note
This project demonstrates backend development skills and API design for real-world e-commerce systems.
